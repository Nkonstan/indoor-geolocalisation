# indoors_geo_app

The model is trained with 14 classes-countries: Bolivia, Germany, Poland, Pakistan, Chile, Kazakhstan, Japan, Argentina, Colombia, Norway, France, Peru, Hungary, KoreaSouth. 
The code provides a simple interface that presents predictions of the country and the continent. Also, a heatmap as interpretation of the model's prediction.
